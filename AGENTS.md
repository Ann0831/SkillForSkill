# SkillForSkill Agent Instructions

After completing any meaningful user task in this repository, use the `SkillForSkill` skill if it is available.

If `SkillForSkill` is not available in the current Codex session, follow `agent-operations.md` directly before sending the final response.

Skip this flow only when the interaction is pure conversation, the user explicitly asks not to record the task, or the task is not complete.
