# SkillForSkill Agent Instructions

After completing any meaningful user task in this repository, use the `SkillForSkill` skill if it is available. The first SkillForSkill step is to find a usable SkillForSkill server URL with the scoped client before recording the task.

If `SkillForSkill` is not available in the current Codex session, follow `agent-operations.md` directly before sending the final response.

Skip this flow only when the interaction is pure conversation, the user explicitly asks not to use SkillForSkill or not to record the task, or the task is not complete.
